    3.

    This is a realistic, steady, one–dimensional ADR model with spatially varying coefficients and a nonhomogeneous source term. Boundary‑value problem (BVP):
    🔗
    \begin{equation*} \frac{d}{dx} \left(D(x)C'(x)\right)-v(x)C'(x)-k(x)C(x) = S(x), \qquad 0 < x < L, \end{equation*}
    \begin{equation*} C(0)=C_0, \qquad C'(L)=0. \end{equation*}
    Coefficient models (realistic spatial variation):
    🔗
    • Dispersion with scale‑dependent dispersivity:
      🔗
      \begin{equation*} D(x) = D_m + \alpha(x)\,v(x), \qquad \alpha(x) = \begin{cases} \alpha_{\text{grad}}\,x, & 0 \le x < x_v,\\ \alpha_L, & x \ge x_v~. \end{cases} \end{equation*}
      🔗
    • Velocity from Darcy kinematics with a mild downstream decrease:
      🔗
      \begin{equation*} v(x)=\dfrac{K_0\,i_0}{n\,(1+\beta x)}~. \end{equation*}
      🔗
    • First‑order decay increasing with distance (e.g., enhanced biodegradation):
      🔗
      \begin{equation*} k(x)=k_0\,e^{\gamma x}. \end{equation*}
      🔗
    • Distributed source that fades with distance:
      🔗
      \begin{equation*} S(x)=S_0\,e^{-\alpha_s x}. \end{equation*}
      🔗
    Default parameters for the numerical experiment (use unless a part tells you to change them):
