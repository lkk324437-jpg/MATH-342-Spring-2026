    Definition 1.

    If \(f\) is a function defined for \(t\geq 0\text{,}\) then the integral \(\mathscr{L}\{f(t)\}=\int_0^\infty e^{-st}f(t)dt\) is the Laplace transform of \(f, \) provided the integral converges.

    Sufficient conditions for existence:
    • \(f\) must be piecewise continuous on \([0,\infty]\)

    Definition 2.

    A function \(f\) is said to be exponential order if there exists constants \(c, M\gt 0,\) and \(T\gt 0\) such that
    \begin{equation*} |f(t)|\leq M e^{ct}, \text{for all }t\gt T. \end{equation*}

    Definition 7.

    If \(F(s)\) represents the Laplace transform of a function \(f(t),\) then \(f(t)\) is the inverse Laplace transform of \(F(s),\)
    \begin{equation*} f(t)=\mathscr{L}^{-1}\left\{F(s)\right\} \end{equation*}

    Remark 11.

    Conceptually, the original DE is transformed to another equation in a different space. Ideally, this new equation is easier to solve in the new space. Finally, the solution in the new space is transformed back to the original space using the inverse transformation.
    There are many other transformations and each one is considering a different space, which also means each one is more suitable for solving some particular types of DEs. This is a typical issue for most of algebraic approaches for solving ODEs or PDEs.

    Theorem 13.

    Suppose \(f, f', \ldots, f^{(n-1)}\) are continuous on \([0,\infty)\text{,}\) \(f^{(n)}\) is piecewise continuous on \([0,\infty),\) and \(f, f', \ldots, f^{(n-1)}, f^{(n)}\) are of exponential order with \(c\) as specified in DefinitionÂ 2. Then for \(s\gt c,\)
    \begin{equation*} \mathscr{L}\left\{f^{(n)}(t)\right\}=s^nF(s)-s^{n-1}f(0)-s^{n-2}f'(0)-\cdots-f^{(n-1)}(0), \end{equation*}
    where \(F(s)=\mathscr{L}\left\{f(t)\right\}.\)
